# dakshck.github.io

So this is my GitHub Pages website repository where I host my personal portfolio.

Initially, it was generated with **v0**, but over time I customized almost everything until it actually felt like my own project.

---

## Things I changed

- Added **live GitHub contributions** and **recently updated repositories**. (Believe me, before this there were just random CSS boxes.)
- Redesigned the footer with Discord links and the giant **dakshck** branding. Every part of it was made by me.
- Added an interactive **"Daksh Chordiya"** text on the Hero section.
- Fixed and redesigned the background because it was completely broken before.
- Reworked the Projects section with a **View More** button and a dedicated page.
- Improved the overall branding with better typography and by removing unnecessary logos.
- Cleaned up lots of UI details throughout the website.
- Spent way too much time staring at code without knowing what was going on... somehow it all worked out in the end.
